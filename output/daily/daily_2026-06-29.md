### 04:00 RedKnot: Efficient Long-Context LLM Serving with Head-Aware KV Reuse and SegPagedAttention

- **자료유형:** 논문

- **저자/발행기관:** arXiv CS.AI

- **링크/DOI:** https://arxiv.org/abs/2606.06256
- **요약 (영문 원문):**
  - RedKnot is a head-aware KV cache management system for large language model serving, addressing the bottleneck of KV cache in AI infrastructure. This system improves resource efficiency and scalability without requiring model retraining or fine-tuning.
  - RedKnot breaks the conventional monolithic KV cache abstraction by decomposing the KV cache along KV heads, enabling uniform support for position-independent KV reuse, prefix KV compression, hot/cold KV separation, and distributed KV placement.
  - RedKnot's head-level decomposition turns the KV cache into a structured memory object, allowing it to adapt to different serving scenarios and attention ranges, and preserving output fidelity.
- **요약 (한국어 번역):**
  - RedKnot은 대형 언어 모델(LLM) 서비스를 위한 헤드-지식(KV 캐시) 관리 시스템으로, AI 인프라에서 KV 캐시의 병목 현상을 해결하고 리소스 효율성을 개선함.
  - RedKnot은 KV 캐시를 KV 헤드별로 분할하여, 위치 독립적 KV 재사용, 접두사 KV 압축, 핫/콜드 KV 분리 및 분산 KV 배치에 대한 균일한 지원을 제공함.
  - RedKnot은 KV 캐시를 구조화된 메모리 객체로 변환하여, 다양한 서비스 시나리오와 주의 거리에서 KV 캐시의 유용성을 구조화함.
