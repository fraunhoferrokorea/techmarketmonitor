# 데일리 리서치 로그 양식 (기사 + 논문 통합)

기사든 논문이든 동일한 양식으로 기록합니다. **자료유형** 필드로만 구분합니다.
하루에 여러 건을 기록한 날은 맨 위에 **"오늘의 요약(Daily Executive Summary)"**을 추가합니다.

**파일명 권장:** `daily_YYYY-MM-DD.md` (Cursor가 날짜순으로 폴더를 읽기 쉽도록)

---

## 파일 상단 메타데이터 (파일 1개 = 하루 1개)

```
날짜: YYYY-MM-DD
기록자:
총 항목 수: N건 (기사 n / 논문 n)
신뢰도 분포: A n건 / B n건 / C n건
```

---

## 오늘의 요약 (Daily Executive Summary)

> 그날 기록한 항목이 **2건 이상**일 때 작성. 1건뿐이면 생략 가능.
> 항목들을 단순 나열하지 말고, "오늘 본 것들이 합쳐서 무엇을 말하는가"를 3~5줄로.

- **오늘의 핵심:** (가장 중요했던 1~2개 포인트)
- **눈여겨볼 신호:** (사소해 보이지만 추세로 이어질 수 있는 것)
- **상충되는 정보:** (있다면 — 서로 다른 소스가 다른 말을 하는 경우)

---

## 항목 기록 (항목당 1블록, 하루에 여러 개 반복)

### [HH:MM 또는 순번] 제목

- **자료유형:** 기사 / 논문 / 보고서(시장조사) / 공식발표(IR·정책) / 기타
- **출처:** (매체명·저널명·기관명)
- **저자/발행기관:** (논문일 경우 저자명, 기사일 경우 매체)
- **발행일:** YYYY-MM-DD
- **링크/DOI:**
- **요약 (3~5줄):**
  - 사실관계 중심. 의견·해석은 마지막 줄에 "(해석)"으로 별도 표시
- **신뢰도:** A / B / C (기준 하단 참조)
- **태그:** #태그1 #태그2
- **비고:** (선택 — 다른 항목과의 연결점, 왜 중요한지)

---

## 태그 분류체계 (월간 보고서 챕터와 매칭)

| 태그 | 의미 |
|------|------|
| #기술 | 신기술, R&D, 기술 발표 |
| #논문 | 학술 연구 결과 (성능, 실험, 방법론) |
| #투자 | 투자 라운드, 밸류에이션, 펀딩 |
| #M&A | 인수합병, 전략적 제휴 |
| #제품출시 | 신제품, 서비스 런칭 |
| #기업동향 | 조직, 인력, 실적, 전략 |
| #규제 | 법안, 정책, 표준 |
| #경쟁 | 경쟁사 비교, 점유율, 포지셔닝 |
| #시장수치 | 시장규모/성장률 추정치 |
| #리스크 | 사고, 논란, 부정적 전망 |
| #전문가전망 | 애널리스트/전문가 예측 |

여러 태그 중복 가능 (예: `#논문` `#기술`)

---

## 신뢰도 등급 기준

- **A (높음):** 피어리뷰 학술지 논문, 1차 보도(공식발표 인용), Tier-1 통신사(Reuters/Bloomberg/AP), 정부·국제기구 통계, Tier-1 시장조사기관(Gartner/IDC/McKinsey)
- **B (중간):** 프리프린트(arXiv 등 동료심사 전), 업계 전문매체, 2차 인용 보도, 기업 자체 발표(IR/보도자료)
- **C (참고):** 익명 소스, 추측성 기사, 단순 재가공 콘텐츠, 미검증 블로그

## 기록 예시

### 09:40 Acme AI, 시리즈 C 2억 달러 유치

- **자료유형:** 기사
- **출처:** TechCrunch
- **저자/발행기관:** TechCrunch 편집부
- **발행일:** 2026-06-15
- **링크/DOI:** https://example.com/article
- **요약 (3~5줄):**
  - Acme AI가 시리즈 C로 2억 달러 유치, 기업가치 12억 달러로 평가됨
  - 주도 투자사는 XYZ Capital, 기존 투자자 ABC Ventures도 참여
  - 자금은 추론 비용 절감을 위한 자체 칩 개발에 사용 예정
  - (해석) 최근 3개월간 동일 분야 투자가 3건째로, 섹터 과열 신호일 가능성
- **신뢰도:** B
- **태그:** #투자 #경쟁
- **비고:** 경쟁사 BetaCorp 동향과 비교 필요

### 14:10 효율적 어텐션 메커니즘에 관한 신규 연구

- **자료유형:** 논문
- **출처:** arXiv
- **저자/발행기관:** Lee et al., Seoul National Univ.
- **발행일:** 2026-06-14
- **링크/DOI:** arXiv:2606.xxxxx
- **요약 (3~5줄):**
  - 기존 어텐션 대비 연산량 40% 절감하는 신규 메커니즘 제안
  - 벤치마크 3종에서 성능 저하 없이 속도 개선 확인
  - 아직 피어리뷰 전 단계(프리프린트)
  - (해석) 실제 프로덕션 적용까지는 추가 검증 필요해 보임
- **신뢰도:** B (프리프린트, 동료심사 전)
- **태그:** #논문 #기술
- **비고:** 다음 달 컨퍼런스 제출 여부 확인 필요
